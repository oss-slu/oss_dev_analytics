# oss_dev_analytics

## Metrics
* Git Logs
  * Lead Time
  * Cycle Time
  * Delivery Metrics
  * Commit Frequency (per author)
* Git Issues
  * Opened Frequency (per author)
  * closed Frequency (per author)
  * Resolution Time
  * PR (Merge Time)
  * WIP
* Surveys
  * Client Satisfaction
  * Team Satisfaction
  * Collaboration
  * Sprint Confidence
  * Goal Achievement
